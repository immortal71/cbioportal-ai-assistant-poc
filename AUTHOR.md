### About This PoC

This proof-of-concept demonstrates:
- Natural language query processing for cancer genomics data
- Real-time integration with cBioPortal public API
- Advanced gene and cancer type detection
- User-friendly web interface for complex data exploration
- Successfully tested with 1,405+ real mutations across 6 genes

### Technical Stack

**Backend:**
- Python 3.8+
- FastAPI 0.109.0
- Uvicorn 0.27.0
- httpx 0.26.0 (async HTTP client)

**Frontend:**
- HTML5, CSS3, Vanilla JavaScript
- Responsive gradient UI design

**API Integration:**
- cBioPortal Public API
- RESTful architecture
- JSON data exchange
